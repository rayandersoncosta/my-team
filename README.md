# In English

## Install & Start

⚠️ Using [Yarn Package Manager](https://yarnpkg.com) is recommended over `npm`.

Install dependencies

```shell
yarn install
```

Start project

```shell
yarn start
```

# Em Português

## Instalar e iniciar
